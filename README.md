# ⚡ NEON SPORTS CMS ⚡

A retro-styled, neon-themed sports content management system built with React, Vite, and Tailwind CSS.

## 🎨 Features

- **Retro Neon Aesthetic**: Vibrant neon colors with glow effects
- **Dashboard Overview**: Quick stats on total, upcoming, and completed events
- **Event Management**: Add, view, and delete sports events
- **Responsive Design**: Works on all screen sizes
- **Grid Background**: Classic 80s-style grid pattern
- **Animated Elements**: Flickering neon effects and pulsing animations

## 🚀 Tech Stack

- React 18
- Vite
- Tailwind CSS
- Custom neon animations

## 📦 Installation

```bash
npm install
```

## 🏃 Development

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
